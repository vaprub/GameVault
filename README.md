# 🎮 GameVault - Менеджер игровых аккаунтов

Безопасное хранилище для ваших игровых аккаунтов и привязанных почт.

## ✨ Возможности

- 🔐 Шифрование AES-256 (через мастер-пароль)
- 📝 Хранение данных аккаунта + привязанной почты
- 🔑 Генератор надежных паролей
- ⏰ Автоблокировка через 5 минут бездействия
- 💾 Экспорт/импорт резервных копий
- 📦 Портативность (можно носить на флешке)
